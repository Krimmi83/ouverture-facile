About

Static-only, multi-game edition. Data model per player:

{
  id, username, email,
  games: {
    smash: { wins, losses },
    mariokart: { wins, losses },
    demineur: { difficulty, score },
    minecraft: { kills, deaths }
  }
}

Legacy data (single wins/losses) auto-migrates to Smash.

Built with ❤️ — no frameworks.
